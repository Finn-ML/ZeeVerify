Authentication & User Management	Login & Signup	Basic login and signup page for users to get access to the app (email & password). Includes forgot password flow, user type selection during signup (Franchisee vs Franchisor vs Browser), and basic form validation.
Authentication & User Management	User Account Settings	Allow users to alter their settings, change password, email, profile editing, notification preferences, and account deletion option.
Authentication & User Management	Stripe ID Verification	Franchisee identity verification using Stripe Identity integration. Includes document upload flow, verification status tracking, and matching uploaded ID to franchise agreement name.
Franchisee Portal	Franchisee Portal	Portal for franchisees to manage their reviews, submit feedback, view verification status, track historical reviews, see franchisor responses, and flag incorrect information for admin review.
Franchisee Portal	AI Sentiment Analysis & Content Moderation	OpenAI integration for automated content analysis. Detects sentiment (positive/negative), graphic content, profanity, and name mentions. Auto-routes content based on analysis results into clean, needs review, or rejected categories.
Franchisor Portal	Franchisor Portal	Allows franchisors to claim profiles, respond to reviews, upload verified performance data (videos, images, documentation), view lead inbox, report reviews, and access analytics dashboard with review trends and sentiment breakdown.
Franchisor Portal	Review Response System	Response composer for franchisors to reply to reviews. Includes response moderation queue, public display of responses, and notification to franchisee when response is posted.
Public-Facing Features	Brand Listings & Directory	Directory of approximately 4,000 franchise brands with search functionality. Displays aggregate metrics visible to all users, claimed vs unclaimed badges, category filtering, and Z Score display.
Public-Facing Features	Comparison & Filtering Tools	Tools allowing users to compare brands side-by-side based on various metrics. Includes multiple filter criteria (industry, location, rating), sort options, and save/share comparisons functionality.
Public-Facing Features	Key Terms & Word Frequency Visualization	Visual display showing how many times specific terms or phrases were mentioned in reviews (e.g., 'terrible support mentioned 52 times'). Includes word clouds, frequency charts, trend tracking, and positive vs negative term categorization.
Public-Facing Features	Z Score Calculation Engine	Algorithm for calculating overall brand value score based on franchisee feedback. Includes real-time calculation updates, historical score tracking, and score breakdown by category.
Public-Facing Features	Request More Information Form (Lead Capture)	Lead capture form for users requesting detailed information about a brand. Includes information gating (unlock detailed reviews after submission), lead notification system, and CRM-style lead tracking.
Admin Features	Admin Dashboard	Central hub displaying key metrics (users, reviews, listings, leads), review volume trends, moderation queue statistics, revenue/subscription tracking, and quick action panels.
Admin Features	Admin User Management	View, search, and manage user accounts. Includes user type management, verification status management, account suspension/deletion, password reset sending, and bulk actions.
Admin Features	Trust & Verification Layer	Admin moderation system for ensuring authenticity and quality of reviews. Includes moderation queue with AI-flagged items, manual review and edit capability, approve/reject workflow, reported content queue, and moderation history logging.
Admin Features	Content Management	Management of terms and conditions, brand listing management (bulk import of 4,000 brands), and category/industry taxonomy management.
Payments & Monetization	Stripe Checkout	Stripe integration for processing listing claim fees. Includes one-time claim fee processing, subscription setup for premium features (lead gen), payment history, and invoice generation.
Payments & Monetization	Lead Routing System	System for distributing leads based on subscription tier. Routes leads to franchisor if subscribed, otherwise to Z Verify team. Includes notification routing and lead status tracking.
Notifications	Notifications (Advanced)	Email notification system using Postmark/SendGrid integration. Includes transactional emails (password reset, verification), new review notifications, lead notifications, moderation outcome notifications, subscription/payment emails, and HTML email templates.
