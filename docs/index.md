## Architecture Summary

```
┌─────────────────────────────────────────────────────────────┐
│                         CLIENT                               │
│  React 18 + Vite + TanStack Query + Tailwind CSS            │
│  UI: Shadcn/ui (48 components) + Custom business components │
└─────────────────────────────────────────────────────────────┘
                              │
                         HTTP + JSON
                              │
┌─────────────────────────────────────────────────────────────┐
│                         SERVER                               │
│  Express + Drizzle ORM + Replit Auth (OIDC)                 │
│  Storage layer pattern for database operations               │
└─────────────────────────────────────────────────────────────┘
                              │
                         Drizzle ORM
                              │
┌─────────────────────────────────────────────────────────────┐
│                        DATABASE                              │
│  Neon PostgreSQL (serverless)                               │
│  11 tables: users, brands, reviews, leads, etc.             │
└─────────────────────────────────────────────────────────────┘
```

---

## Key Entities

| Entity | Purpose |
|--------|---------|
| **User** | Account with role (browser/franchisee/franchisor/admin) |
| **Brand** | Franchise listing (~4,000 entries) |
| **Review** | Franchisee feedback with ratings |
| **Lead** | Prospective buyer inquiry |

## API Endpoint Groups

| Group | Auth | Description |
|-------|------|-------------|
| `/api/auth/*` | Mixed | Authentication flow |
| `/api/brands/*` | None | Brand directory |
| `/api/reviews/*` | Required | Review operations |
| `/api/leads` | None | Lead capture |
| `/api/franchisee/*` | Required | Franchisee portal |
| `/api/franchisor/*` | Required | Franchisor portal |
| `/api/admin/*` | Admin | Administration |
